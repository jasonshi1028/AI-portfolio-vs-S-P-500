# AI Portfolio vs S&P 500

This project studies whether a basket of AI-related companies outperformed the S&P 500 during 2023.

## Research Question
Did AI-related stocks generate excess returns relative to the S&P 500, and did that outperformance come with higher volatility?

## Method
- Selected a basket of AI-related companies
- Built an equal-weight AI portfolio
- Compared returns and volatility against the S&P 500
- Calculated a simple risk-adjusted performance measure
